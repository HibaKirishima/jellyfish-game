# jellyfish-game

##目標: 鍵を集めて時間内にドアのロックを解除する 

##コントロール: 矢印を使って上、下、右、左に移動します 

##スクリーンショット

##Goal
Collect the keys and unlock the doors in time record

##Controls
use arrows to go up,down,right,left
